# MwManger Agent 분석 보고서

이 보고서는 `MwManger Agent` 소스 코드를 분석하여 프로젝트의 목적, 구조, 핵심 로직 및 보안 기능을 상세히 설명합니다.

## 1. 프로젝트 개요 (What is this?)

**MwManger Agent**는 서버로부터 원격 명령을 수신하여 실행하는 **Java 기반의 원격 관리 에이전트**입니다.

*   **주요 역할**: 서버(Management Server)에 접속하여 명령을 받아오고(Polling), 쉘 스크립트 실행, 파일 다운로드 등의 작업을 수행합니다.
*   **통신 방식**: HTTP(Polling) 및 Kafka(Message Queue)를 지원합니다.
*   **보안**: 강력한 보안 검증(Injection 방지, 경로 탐색 방지 등) 기능을 내장하고 있습니다.
*   **기술 스택**: Java 8, Gradle, Apache HttpClient, Kafka Client.

## 2. 동작 원리 (How it works?)

에이전트는 다음과 같은 생명주기(Lifecycle)를 따라 동작합니다.

1.  **시작 (`MwAgent.java`)**: 에이전트가 시작되면 설정을 로드하고 `STARTING` 상태로 진입합니다.
2.  **등록 (`PreWork.java`)**: 서버에 "나 시작해요"라고 알리고 등록 절차를 수행합니다.
    *   `noticeStart()`: 서버에 시작 알림.
    *   `registerMe()`: 최초 실행 시 에이전트 정보를 서버에 등록.
3.  **초기 작업 (`FirstWork.java`)**: Kafka 설정 등 초기 구동에 필요한 명령을 받아 수행합니다.
4.  **메인 루프 (`MainWork.java`)**: 무한 루프를 돌며 서버로부터 명령을 주기적으로 가져옵니다(`suckCommands`).
5.  **명령 실행 (`OrderCallerThread`)**: 받아온 명령을 별도의 스레드에서 실행합니다.

## 3. 핵심 컴포넌트 분석

### 3.1 메인 진입점 (`MwAgent.java`)
프로그램의 시작점입니다. `LifecycleManager`를 통해 에이전트의 상태(Created -> Starting -> Running)를 관리하며, `PreWork` -> `FirstWork` -> `MainWork` 순서로 로직을 제어합니다.

### 3.2 작업 처리 (`MainWork.java`)
에이전트의 심장부입니다.
*   `doAgentWork()` 메서드에서 `while` 루프를 돕니다.
*   `suckCommands()`: HTTP GET 요청으로 서버에서 명령 목록을 가져옵니다.
*   `processCommands()`: 가져온 명령을 `ExecutorService`(스레드 풀)에 넘겨 비동기로 실행합니다.

### 3.3 명령 실행 (`order/ExeShell.java`)
실제 쉘 명령을 실행하는 클래스입니다.
*   `execute()`: 명령 실행의 진입점입니다.
*   보안 검증: 실행 전 `SecurityValidator`를 통해 경로, 파일 확장자, 인자값 등을 엄격하게 검사합니다.
*   `Runtime.exec()`: 검증된 명령을 OS 레벨에서 실행합니다. (Windows/Linux/AIX/HPUX 지원)

### 3.4 보안 검증 (`security/SecurityValidator.java`)
이 프로젝트의 가장 중요한 특징 중 하나인 보안 모듈입니다.
*   **Command Injection 방지**: `;`, `|`, `&&`, `$()` 등 위험한 쉘 메타 문자가 포함된 명령을 차단합니다.
*   **Path Traversal 방지**: `../` 와 같이 상위 경로로 이동하려는 시도를 차단합니다.
*   **위험 명령어 차단**: `rm -rf`, `mkfs`, `reboot` 등 시스템에 치명적인 명령어 실행을 원천 봉쇄합니다.
*   **SSRF 방지**: `localhost`나 내부 IP 대역으로의 접속을 차단합니다.

## 4. 폴더 구조 설명

```
src/main/java/mwmanger/
├── MwAgent.java              # 프로그램 시작점 (Main)
├── agentfunction/
│   ├── PreWork.java          # 서버 등록 및 초기화
│   └── MainWork.java         # 메인 무한 루프 (명령 폴링)
├── common/
│   ├── Common.java           # HTTP 통신 등 공통 유틸리티
│   └── Config.java           # 설정 파일 로드 및 관리
├── order/                    # 명령 실행 구현체들
│   ├── ExeShell.java         # 쉘 실행
│   ├── DownloadFile.java     # 파일 다운로드
│   └── ...
├── security/
│   └── SecurityValidator.java # 보안 검증 로직
└── vo/                       # Value Objects (데이터 객체)
```

## 5. 실행 방법

**빌드:**
```bash
./gradle-8.5/bin/gradle build --no-daemon --offline
```

**실행 (Fat JAR):**
```bash
java -jar build/mwmanger-all-1.0.0.jar
```

## 6. 요약

이 코드는 **보안성이 강화된 엔터프라이즈급 원격 관리 에이전트**입니다. 단순히 명령을 실행하는 것을 넘어, 에이전트의 라이프사이클을 관리하고 악의적인 명령 실행을 방지하기 위한 안전장치가 잘 갖춰져 있습니다.
