# MWAgent 서비스화 프로젝트 현황

## 1. 프로젝트 개요
**MWAgent**를 Windows 서비스로 등록하여 시스템 부팅 시 자동으로 백그라운드에서 실행되도록 구성하는 프로젝트입니다.

## 2. 현재 구성 상태

### 디렉토리 구조 (`c:\Users\SYSADMIN\MWAgent`)
- **실행 파일**: `mwagent-service.exe` (Service Wrapper)
- **Java 애플리케이션**: `mwagent.jar`
- **설정 파일**: `mwagent-service.xml`
- **로그 및 기타**: `lib/`, `script/`, `log/`

### 서비스 설정 (`mwagent-service.xml`)
- **서비스 ID**: `MWAgent`
- **서비스 명**: `MWAgent Service`
- **실행 명령**: Java VM을 통해 `mwagent.MwAgent` 클래스 실행
- **메모리 설정**: Min 256MB / Max 256MB
- **환경 변수**: USER, HOSTNAME, DOMAIN_NAME

## 3. 진행 상황

### ✅ 완료된 작업
- [x] 서비스 래퍼(Wrapper) 및 실행 파일 배치
- [x] 서비스 설정 파일(xml) 작성 및 환경변수 구성
- [x] 인증 문제 해결 (Double Slash 제거, 새 토큰 적용)
- [x] 서비스 구동 완료

## 4. 서비스 상태 확인 방법

### PowerShell
```powershell
Get-Service MWAgent
Get-Content "C:\Users\SYSADMIN\MWAgent\log\mwagent.0.0.log" -Tail 20 -Wait
```

### CMD
```cmd
sc query MWAgent
```

## 5. Java 직접 실행 (서비스 미사용 시)
*   **`start_mwagent.bat`**: 에이전트 시작 (콘솔 모드)
*   **`stop_mwagent.bat`**: 에이전트 안전 종료
