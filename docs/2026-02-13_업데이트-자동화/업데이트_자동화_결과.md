# MWAgent 업데이트 및 기능 개선 결과 (v0.9.15)

요청하신 `mwmanger-0.9.15.jar` 업데이트와 함께, **자동화 스크립트**, **재시작 편의 기능**, **통합 로깅 시스템**을 구축하고 **불필요한 파일을 정리**했습니다.
또한 **v0.9.15에서 변경된 Main Class** 설정도 적용 완료했습니다.

## 주요 변경 사항

### 1. Jar 파일 표준화 및 버전 정보
- **파일**: `mwagent.jar` (고정 이름 사용)
- **현재 버전**: `0000.0009.0015` (업데이트 완료)
- **설정 변경**: Main Class가 `mwmanger.MwAgent`에서 **`mwagent.MwAgent`**로 변경됨

### 2. 스크립트 개선 및 신규 생성
모든 스크립트는 `log\script_history.log`에 실행 이력을 남깁니다.

| 스크립트 | 설명 | 로깅 내용 |
| --- | --- | --- |
| `start_mwagent.bat` | 에이전트 시작 | 시작 시간, 성공/실패 여부 |
| `stop_mwagent.bat` | 에이전트 중지 | 중지 시간, 성공/실패 여부 |
| `run.agent.bat` | 자동 업데이트 및 재시작 | 업데이트 전 백업 정보, 결과 |
| `restart.bat` | 간편 재시작 | 재시작 시도 시간, 결과 |

### 3. 디렉토리 정리 및 로그 중앙화
- **로그**: `log` 폴더로 모든 로그 파일 이동 및 통합 기록
- **파일 정리**: 구버전 Jar, 백업 파일 등을 `_old` 폴더로 이동
