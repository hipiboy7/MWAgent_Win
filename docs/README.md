## 운영 참고사항

### JAR 업데이트 흐름 (자동)
```
MWAgent → 서버에서 mwagent.jar 다운로드 (덮어쓰기) → run.agent.bat → 서비스 재시작
```

### JAR 업데이트 흐름 (수동, 파일잠금 이슈 발생 시)
```
새 JAR을 new_mwagent.jar로 복사 → run.agent.bat → restart.bat이 감지하여 백업 → 교체 → 재시작
```

### 파일 잠금 참고
- Java는 JAR 로드 후 파일 핸들을 해제하므로, 실행 중 덮어쓰기 가능 (Linux/Windows 동일)
- Windows에서 파일 잠금 발생 시, `new_mwagent.jar` 방식으로 전환 가능 (restart.bat에 로직 내장)
